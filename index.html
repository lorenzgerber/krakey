Krakey API

Krakey is an REST API to process Kraken.report files into sankey plots.

To convert a kraken.report file, make a POST request for example using curl:
curl -k -X POST -F “sampleFile=@example.report” “krakey.info/upload”

The file will then be converted into a html sankey plot using sankeyD3 from F. Breitwieser. To obtain the sankey plot, perform a GET request: wget krakey.info/sankey/example.html
